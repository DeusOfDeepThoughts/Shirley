# 📚 Shirley

## ✨ 功能：
1. 通过巧妙的提示词工程，给 AI 立好“规矩”，让它老老实实地按照你的要求精读论文。

## 🚀 使用方法：
1. 下载后直接点击运行，即刻开箱即用。
2. 详细用法指南：
   * 新同学强烈建议先设置自己的自定义规则，调教出最懂你的助手。
   * Shirley 在自定义规则里已经内置了一套默认规则，如果你懒得动手，它就会自动按默认规则跑起来。
   * 这里的灵魂是 **“重点关注”** 模块——首页和自定义规则页都有它。如果你在首页填了关注点，Shirley 会优先听从首页的安排；否则就乖乖使用自定义规则页的默认设置。
   * “重点关注”模块支持你上传文档、图片等形式的关注点，例如你可以传一张 `focus.png`，里面写着“实验流程、公式推导”等等，Shirley 都能读懂。
   * 📄 Shirley 要求你至少上传 1 篇待精读的文档，这是开启任务的钥匙。
   * 一切就绪后，点击生成任务包，一个 `任务包.zip` 就会自动下载到本地。
   * 对于 ChatGPT 这类支持 zip 格式的 AI，直接把 zip 扔给它就好；如果遇到 Kimi 等“不吃 zip”的选手，只需解压后把里面所有材料一股脑上传即可。
   * 🎉 最后，挑选你喜欢的任意一款 AI 来分析任务包，生成专属于你的精读报告。

## 💡 优势：
1. 精读论文时，你可以随心选择任何你青睐的 LLM，完全不被绑定。
2. 💰 精打细算，把好钢用在刀刃上：
   * 本地 agent 在调用本地工具（比如绘图）时很拿手，但 token 额度往往有限。如果只是精读一篇论文，真的值得动用宝贵的 token 去做一件在线模型就能轻松搞定的事吗？
   * API 调用方式虽然能在对话框里便捷安排任务，可对于精读量山大的学生党来说，这烧钱速度着实肉疼。
   * 要是直接把在线模型入口塞进 Shirley，操作流程确实简化了，但账号安全风险也随之而来。
   * 🎓 Shirley 当前的设计思路，就是真心想帮广大学生群体省钱、省 token，把有限的资源精准用在刀刃上。哪怕你手头只有免费在线模型，靠 Shirley + 免费 AI 的组合，依然能享受到相当优质的精读服务。
3. 🛠️ Shirley 默认就配好了一套全面又严谨的约束项：角色设定、低幻觉、重点关注、输出要求……当然，你完全可以按自己的喜好随意修改，把它调教成你的形状。
